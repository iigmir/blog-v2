作者

I am a hacker because I write hacks.

稍微正式點？

我是一位軟體開發者，主攻前端。不過由於邏輯比設計好，對後端也有理解。專長是 API 互動與邏輯問題。

最擅長的語言是 JavaScript。對 PHP 與 TypeScript 也有理解。很久以前是 Python 與 Ruby 起家。不過現在比較少碰。

設計比較偏好簡單的風格、程式則越少、越純粹、越像原本的寫法越好。所以模塊會切得很細、而且。前端也比較偏好寫起來像原生 HTML, CSS, JavaScript 的 Vue 框架。我也對 React 有研究就是了。

程式以外的東西

程式以外的興趣有電腦、閱讀、寫作、外語。是個喜歡打遊戲的 ACG 迷。最喜歡的遊戲是 Democracy、動畫則是白箱。

嗯對，我對當今政治與社會有點想法。不過先在此打住吧。

寫過啥？做過啥？

嗯。Github 奉上。
